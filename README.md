## About

Puts grime into your pipes

![Demo](./media/demo.gif)

## Features

- Slow
- Follows unix philosophy

## Example usage

```sh
ls | grime
```

```
sudo find / | grime 2
```

```sh
ps -A | grime 3
```

```sh
pacman -Qe | grime 4
```

```sh
journalctl | grime 5
```
